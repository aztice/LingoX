# LingoX
A Coding Language based on Python.

# Usage
Run Your LingoX Code
`python LingoX.py -run $filename.py`

# Function
> log()

Example Code:
`log("Hello World")`
Output:
`Hello World`

> input()

Example Code:
`YourInput = input("Enter>")`
OutPut:
`Enter>`

> Math Function

Example Code:
```
a = 1+1
log("The sum of a is {a}")
```
OutPut:
`The sum of a is 2`
